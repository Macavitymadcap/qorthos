Book Style Pattern Library

A CSS theme for markdown-derived HTML

This pattern library showcases a CSS theme designed to give HTML content derived from markdown the visual appearance of a traditional printed book. It features carefully selected typography, spacing, and color palettes that mimic the aesthetics of print publishing.

The theme includes a light mode resembling aged paper that provides an elegant reading experience reminiscent of traditional books.

Colour Palette

Main Colours

This carefully selected colour palette is designed to mimic traditional book printing while ensuring good readability. The light mode features soft, warm paper tones for a comfortable reading experience.

Typography

Font Families

Body Text: Libre Baskerville - A classic serif typeface perfect for long-form reading with excellent readability.

Headings: Playfair Display - An elegant display serif with beautiful contrast that provides visual hierarchy.

Technical: Source Sans 3 - A clean sans-serif for technical content.
